🇺🇸 USA Symbol — Educational Website About U.S. State Symbols

https://usasymbol.com

USA Symbol is an educational project that provides clear and well-structured information about official U.S. state symbols, including:

state mottos

nicknames

state birds

flowers

trees

flags

additional cultural and historical facts

The goal of the project is to create a clean, fast, and accessible resource for students, teachers, and anyone interested in American civics.

🛠 Tech Stack

ASP.NET Core MVC

C#

Razor Views

Tailwind CSS

SQLite

YAML-based content structure

📌 Current Status

The project is actively being developed and expanded.
More states and symbol categories are added weekly.
